### Environment

* node v10.15.3
* npm v6.4.1
* ant v3.18.1
* react v16.8.3
* mobx v5.9.4

### Quickstart

```bash
# Start project
npm start
# Build and package project
npm build
# If you are using yarn
yarn start
yarn build
```
